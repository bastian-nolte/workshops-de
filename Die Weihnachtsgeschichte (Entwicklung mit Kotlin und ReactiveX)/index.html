    fun `gib Futter`(menge: Int): Unit { // void in Java
        println("Wuff… wedel…") // system.out.println
    }
}

Lesson Klassen > Implementierung von Interfaces

https://kotlinlang.org/docs/reference/classes.html 
class Terrier : Hund {
    override val fellfarbe: String
        get() = this.fellfarbe

    override var alter: Int
        get() = this.alter
        set(value) {
            this.alter = value
        }

    override fun `gib Futter`(menge: Int) {
        println("wuff…")
    }
}

Lesson Versiegelte Klassen (sealed class)

https://kotlinlang.org/docs/reference/sealed-classes.html

Versiegelte Klassen Versiegelte Klassen werden zur Darstellung eingeschränkter Klassenhierarchien verwendet, wenn ein Wert einen der Typen aus einer begrenzten Menge haben kann, aber keinen anderen Typ haben darf.

sealed class Haustier(open val alter: Int)

data class Katze(val fellfarbe: String, override val alter: Int) : Haustier(alter)
data class Kaiman(val gefresseneGnus: Int, override val alter: Int) : Haustier(alter)

fun `beschreibe Haustier`(haustier: Haustier) = when (haustier) {
    is Katze -> println("Der Fell der Katze ist $haustier.fellfarbe.")
    is Kaiman -> print("Der Kaiman hat ${haustier.gefresseneGnus} Gnus gefressen.")
}

Story Bööööse Geschenke

Um den Bürgern den Spass am Fest zu nehmen, plant der Gruusige in einer Fabrik Geschenke zu produzieren, die nicht richtig funktionieren oder die Einwohner auf andere Art verärgern.

    Definiere eine Sealed Class mit folgenden Eigenschaften
  • name: String
  • geschlecht: Geschlecht
  • beschreibung: String
    • …und schreibe mindestens 4 Klassen mit Defaultwerten die jeweils eine Klasse von Geschenken repräsentieren.

    Tips / Anmerkungen
  • Erstelle für das Geschlecht (WEIBLICH, MAENLICH, SACHLICH) eine Enumeration.
  • Die Klassen sollen jeweils mit parameterlosem Constructor erzeugt werden können.
